What is a Function? ‚öôÔ∏è

A **function** is a relationship where every input has exactly one output. Think of it like a machine: you put in a number (the **input** or $$x$$), the machine does some work, and a single number comes out (the **output** or $$y$$). We often write functions using **function notation**, like $$f(x)$$ which is read "f of x".

The **domain** is the set of all possible input values ($$x$$). The **range** is the set of all possible output values ($$y$$).

When we graph a function, we plot the pairs of inputs and outputs ($$x, y$$) on a coordinate plane. The graph is a visual representation of the function's behavior.
