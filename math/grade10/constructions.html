What are Geometric Constructions?

**Geometric constructions** are a set of rules for drawing geometric shapes and figures using only two tools: a **compass** and an unmarked **straightedge**. You can think of it as pure geometry, where you use a compass to draw circles and arcs and a straightedge to draw straight lines.

There are several fundamental constructions you should know:

  • **Perpendicular Bisector**: A line that cuts a segment in half at a $90^\circ$ angle.
  • **Angle Bisector**: A line that divides an angle into two equal parts.
  • **Copying a Segment**: Creating a new line segment with the exact same length as an existing one.
  • **Copying an Angle**: Creating a new angle with the exact same measure as an existing one.
