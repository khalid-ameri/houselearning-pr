What is a System of Linear Equations? üßê

A **system of linear equations** is a set of two or more equations with the same variables. The solution is a point (an x and a y value) that satisfies all equations in the system. Graphically, this is the point where all the lines intersect.

This tutorial will cover the three main methods for finding these solutions: **graphing**, **substitution**, and **elimination**.
