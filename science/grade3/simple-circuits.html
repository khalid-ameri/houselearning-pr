What is a Simple Circuit? ü§î

A **circuit** is a path that electricity can travel on. It's like a tiny racetrack for electricity! To make a circuit, you need a few things:

* A **power source**, like a battery. It gives the electricity the energy to move. * A **conductor**, like a wire. It's the path the electricity travels on. * A **load**, like a light bulb. It uses the electricity to do something, like make light!

For a circuit to work, the path has to be closed. If there is a break or a gap, the electricity can't go anywhere, and the light bulb won't light up!
